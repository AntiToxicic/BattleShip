
-----------------------------------------
    Данный документ является настоящей документацией к проекту BattleShip. Здесь описаны
все методы, что они делают и зачем они нужны.
-----------------------------------------


    BattleShip.cs

            Данный класс является главным. Отсюда начинается работа программы. Он запускает 
        генерацию кораблей, затем начинается бой.


    OutPut.cs 

            Данный класс явлется графическим представлением полей с кораблями для консоли.


        Game_Name

            Метод, который отрисовывает название игры


        DrawBattleFields

            Метод, который отрисовывает поле с кораблями игрока и поле противника


        DrawPlayerField

            Метод, который отрисовывает только одно поле


        DrawAimAndBattleFields

            Метод, который отрисовывает оба поля, но при этом добавляет прицел с переданными координатами


        DrawRules

            Метод отрисовывает правила для расстановки кораблей


        DrawRules2

            Метод отрисовывает привила для стрельбы по полю противника


    PlayersData

            Класс организует доступ к данным игрока, к его полями, короблям, также содержит объекты классов для 
        заполнения данными

        
        Generate

            Метод вызывает другой метод для создания кораблей с параметрами 2 игроков

        
        CreateFleet

            Метод вызывает конструктор для создания кораблей с учетом переданных ему данных


        CreateFleet (Перегрузка)

            Метод вызывает конструктор для создания кораблей с учетом переданных ему данных
            в этом методе отсутствует возможность заполнения кораблей вручную, а также добавление
            созданых кораблей на поле боя

        
    Client

        Класс создан для хранения в себе объктов Флота и поля

    
    Fleet

        Класс содержит в себе список для хранения объектов кораблей

    
        IsAliveFleet

            Метод возвращает состояние флота - жив он или мертв
    

    BattleField

        Класс содержит в себе матрицу и ее размер - размер карты

        
        BattleField (Конструктор)

            Заполняет матрицу символами обозначающие пустоту


        MapSize (get)

            Возвращает размер матрицы


    Ship

        Класс является шаблоном для создания коробля с его длиной, ориентацией, координатами.
        Также класс возвращает эти значения


        Ship (Конструктор)

            При создании объекта с данными эти данные копируются в глобальные переменные для хранения


        Damage

            Метод уменьшает значение, отвечающее за количество жизней у короблям


        IsAliveShip

            Метод возвращает состояния коробля - жив он или мертв


        X (get)

            Возвращает координату по оси X

        
        Y (get)

            Возвращает координату по оси Y


        lenghtShip (get)

            Возвращает длинну коробля


        typeShip (get)

            Возвращает тип коробля


    Constructor

        Класс отвечает за генерацию короблей


        AddShip

            Метод добавляет в переданный ему список корабль с данными из глобальных переменных

        
        SetShip

            Метод изменяет переданную ему матрицу по переданным данным. Простыми словами устанавливает
            корабль на поле боя


        findPlace

            Метод ищет место для постановки конкретного коробля со значениями полученными от игрока.
            Далее он передает эти значения в метод SetShip

        
        RandomPlace

            Метод ищет место для постановки конкретного коробля случайным образом, далее он передает
            эти значения в метод SetShip

        
    Checker

        Класс содержит в себе один метод и служит только для того, чтобы проверить можно ли ставить корабль
        на поле


        IsFreePlace

            Метод проверяет можно ли поставить конкретный корабль на поле


    TheBattle

        Класс для реализации боя между игроками


        TurnTurn

            Метод передает очередность хода игрокам до тех пор, пока кто-нибудь из них не выйграет

        
        PlayerTurn

            Метод выбирает место для выстрела по полю противника с данными, полученными от игрока

        
        BotTurn

            Метод выбирает место для выстрела случайным образом по полю противника

        
    BattleManager

        Класс проверяет прильность установок меток на поле боя - попадание, промах, убийство.
        Устанавливает метки


        CheckShot

            Метод проверяет координаты, чтобы установить попадание это или промах
            Если фиксируется попадание по короблю и у него не осталось жизней вызывается метод Explode


        Hit

            Метод устанавливает метку попадания по переданным координатам на переданное поле

       
        Miss
       
            Метод устанавливает метку промаха по переданным координатам на переданное поле

        
        Explode

            Метод устанавливает метки взрыва на корабле и вокруг него


    LenghtShip

        Перечисление, которое содержит в себе возможные длинны короблей

    
    Markers

        Перечисление, которое содержит в себе возможные типы меток


    TypeShip    

        Перечисление, которое содержит в себе возможные типы короблей (горизонтальные или вертикальные)
